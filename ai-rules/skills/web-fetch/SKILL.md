name: web-fetch
description: Fetches web content as Markdown. Use for documentation, articles, and reference pages.
---

Fetch web pages from the terminal using [trafilatura](https://github.com/adbar/trafilatura).

## When to use

- User asks to read, fetch, or analyze a URL
- You need to read documentation, tutorials, or resources

## Basic usage

```bash
trafilatura --markdown -u https://example.com
```
